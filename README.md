# JavaScript – Fundamentos y DOM

Este repositorio contiene ejercicios prácticos de JavaScript enfocados en:

- Programación funcional (map, filter, reduce)
- Manipulación del DOM sin frameworks
- Manejo de estado
- Renderizado dinámico
- Persistencia con localStorage
- Arquitectura de componentes en JavaScript puro

## Estructura

- `01-funcional`: ejercicios de arrays y funciones puras.
- `02-dom-basico`: manipulación directa del DOM y eventos.
- `03-dom-componentes`: componentes autocontenidos con estado y persistencia.

## Objetivo

El objetivo de este repositorio es documentar mi proceso de aprendizaje y
construir una base sólida antes de avanzar a frameworks como React.

## Tecnologías

- JavaScript (ES6+)
- HTML
- DOM API
- localStorage
